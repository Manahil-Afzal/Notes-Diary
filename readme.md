## 📌 Project Overview

**Name:** Smart Notes App  
**Type:** Full Stack Notes Management System  

The Notes App is a modern full-stack web application that allows users to create, edit, and delete color-coded notes in a clean and interactive UI. Each note is stored in a database and persists across sessions, ensuring data is never lost.

This project demonstrates how a real-world CRUD application is built using a **separation of frontend and backend architecture**, with a responsive UI and scalable backend API.

---

## 🎯 Goals of the Project

- Build a full-stack CRUD application from scratch  
- Understand frontend-backend integration  
- Implement REST APIs using NestJS  
- Store and manage data using MongoDB  
- Create a responsive and user-friendly UI  
- Learn real-world project structure used in production  

---
